# Правила задания размеров

Основные правила по использованию размеров текста или отступов из UI-Kit доступны в главе [Дизайн](?path=/docs/документация-дизайн)

При использовании динамического скейлинга в компонентах размеры надо указывать особым образом:
* если надо задать размер вручную, в css, использовать хэлпер `calc-size`;
* если надо задать размер через js, использовать хук `useBaseSize` (стоит использовать только в тех местах, где нужен динамический скейлинг).

## Динамический скейлинг
Динамический скейлинг — механизм изменения размеров шрифтов, отступов и размера элементов.

Управлять динамическим скейлингом можно с помощью специального компонента `BaseSizeProvider`.
